 # ProShop - A MERN Stack gadget e-commerce website:

## Technologies
#### Frontend - HTML, CSS, React.js, React Bootstrap, Redux
#### Backend - Node.js, Express.js
#### Database - MongoDB
 
<br>

<ol>
